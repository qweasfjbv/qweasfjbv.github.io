## BOJ 신기한 소수(2023)

<https://www.acmicpc.net/problem/2023>

### Answer

```cpp
#include <iostream>
using namespace std;

int N;
int cnt = 0;
// 첫번째 숫자는 소수만 가능
int start[4] = { 2, 3, 5, 7 };
// 두번째 부터는 홀수만 가능
int arr[6] = { 1, 3, 5, 7, 9 };

// 소수라면 true 반환
bool Check(int n) {

	if (n % 2 == 0) return false;

	// n은 홀수이기 때문에 2의배수로 확인하지 않아도 됨.
	for (int i = 3; i * i <= n; i += 2) {
		if (n % i == 0) {
			return false;
		}
	}

	return true;

}


void Count(int n) {

	if (cnt >= N-1) {
		cout << n << endl;
		return;
	}



	int ch = n * 10;
	for (int i = 0; i < 5; ++i) {
		ch += arr[i];

		if (Check(ch)) {
			cnt++;
			Count(ch);
			cnt--;
		}
		ch -= arr[i];
	}

	return;

}

int main() {

	cin >> N;
	
	for (int i = 0; i < 4; i++) {
		Count(start[i]);
	}

	return 0;
}
```

소수를 판별할 때 체를 사용하지 못하도록 범위가 최대 8자리, 메모리 제한은 4MB 였다.
그래서 최대한 실행시간을 줄이기 위해서 첫번째 숫자로는 소수만, 그 뒤로는 홀수만 오도록 배열을 통해 구현했다.
